Label Widget 

syntax
label (user-defined) = Label(cibtauber,text = "value", opt = "value")
Label() built-in

arguments AFTER container:

bg or background, fg or foreground, command, image, width, height, text, font=('font-name', size)

fg is text color
bg adds background color to existing items on the screen (not the literal bg)
Example: 

mylabel = Label(root,text = "I am a label widget")

.pack() Aranges and displays the label widget within a parent widget (root), else the label widget cannot show itself. 

Default Behavior: centered, columns
